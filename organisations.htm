Overview

 

Contact details and a profile of your donor, prospect or grantee organisation can be captured here.

See how to capture, save and edit information. Links can be added to other profiles on the database.

 

Tips and best practice

DO enter as much accurate information as you can as this will save you time later.

DO check whether the organisation is already on the system so as to avoid duplications.

DO add acronyms for quick access to the organisation.

 

Topics

 

       Add an organisation

When capturing a new organisation, separate linked profiles will be created for the person and the organisation.

 

1.On your Home dashboard, select the Organisations menu.

Check whether the organisation you want to add is on the system by entering the first few letter of the name into the Search for an organisation box.

 

 

If there are no matching results, select Add new organisation. Enter as much information as you have, and make sure it is accurate.

•Give the organisation an acronym.

•Enter the primary contact person's title, initials and names.

 

Tips to avoid duplication.

•If both the organisation and the person are new, add all the details here, and a new organisation profile as well as a new person profile will be added.

•If the person is already on the system as a contact, leave this blank, and after you have saved the organisation, add a link to that person's profile.

•If you do enter a name here, and when you try to save you receive this message:

oSelect Yes to add a new person profile. This may simply be a case of two different people with the same name.

oSelect No to go back and remove the name. The existing person can then be linked after the organisation has been saved.

oThe third option is to select Cancel.

If you find that there are duplicate people or organisations, you can merge these profiles to ensure that your contacts remain uncluttered.

 

 

2.Once you have entered the basic information, select Save and close if you have completed the entry, or Save and stay on page  if you want to add more details about the organisation.

 

 

Select Save and close once you have completed your entry.

 

3.You can add more persons in DevMan and link them to the organisation. See Add a Person.

 

 

       View an organisation profile

On your Home dashboard, select the Organisation menu, enter the first few letter of the organisation and then select the one you want to view from the drop-down list to open the profile page.

Example of an organisation's profile summary page.

 

You can select whether you would like the Profile summary page or the Edit page to be your default. Click here to see how.

 

       Edit an organisation profile

Review and change the details of an organisation, or add additional information.

Open the organisation's profile summary page, and select Edit.

 

Organisation profile - summary page

 

Organisation edit page

 

Update the profile as needed, then select Save > Save and close to go back to the profile summary page.

To continue working, select Save > Save and stay on page.

You can also select Summary to go back to the organisation summary page.
