Overview

 

DevMan's report and analytics platform provides rich search and reporting for all areas.

Draw statistical high level summaries and graphs, track performance metrics, make sense of your data and gain insights faster.  

Multiple level filtering capabilities allow you to generate tailored reports, create unique groups and segment data in almost any way you need.

 

Topics

 

Features

Search functions are available on all DevMan menus.

The type of information you are looking for determines where to start your search.

•Search as you type - SAYT.

•Search on common fields.

•Advanced search on a combination of filters.

•Saved filters.

•Custom exportable reports.

•Publish reports.

•Draw statistics from key fields.

•Excel integrated reports to build pivot tables and charts.

 

Please see the 'Search & report' quick guide.
