Event grid view

 

1.Enter the Event name and the date of your event.  Choose a Type of event e.g. Fundraising, Marketing, Graduation, Training, Team Building etc. Ask your Devman administrator should you need to add other event types.

 

Add a new event

 

 

 

 

2.If your guest's details already exist on the database, click Add > Add contacts using search page, to compile your guest list.

3.Search for your contacts and select them. You can select a group by clicking Select > All or you may select them individually.

   Once your contacts have been selected, click Add to event. Click Back.

 

Click here to view

 

4.If their details do not exist, and you wish to add them,  you can do this directly from the Event details page, by clicking Add > Add a new guest not yet on the database.  You will be able to Add a person.

 

5.Once your guest list is complete, Click Send > Email / SMS to invite them to your event.

Optional Extras:

 

The following options do not come standard, but if you would find them useful, we can enable these functions for you. Contact: support@kidzafrica.co.za.

 

•Time – this is a free text field to specify a time for the event

•Venue –  this is linked to an organisation profile, so you can use the same venue more than once, and with a whole profile dedicated to the Venue, contact details, addresses, contact people etc can all be captured too.

•Summary – this is for a short description or notes on the event
