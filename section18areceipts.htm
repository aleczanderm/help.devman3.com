SARS 2023 section 18A receipt format

 

New field requirements

The following new fields required by SARS are included in the base s18A receipt template available in the system:

•Donor name / Legal name of entity

•Tax number

•ID number / Passport number / Organisation registration number

•Cell number

•Phone number

•Email address

•Address

To begin using this template, simply clone it and apply your branding.

See the video tutorial below for more detail.

 

How to use the sample template on DevMan

•SARS has published a sample layout for the s18A receipt.

•We have added a SARS-compliant template that can be customised.

•Find the sample template by going to Finance > 18A receipts > Setting > Templates > Print.

•Click on the template to open, Save a copy, then add your own branding and organisation details.

•See the video tutorial below for more detail.

Tax details 2023 - Self-service update

 

Send link to self-service update

•Go to the new SARS detail menu under the Finance tab to see donors identified as needing to update their details.

•Send a bulk email to these donors using a template which includes wording with this link: ../../update/taxdetails/?t=@Token

•NOTE:

oThe Data source must be bulk communication and the Type SARS detail.

oWhen adding the link to the template, link the wording by clicking on the chain link and make sure that the protocol is set to <other>. (See the tutorial below.)

•For new donors, you can set up a workflow so that the link will be sent to them automatically or add it to your thank you email template.

•If you receive a donor's tax details via email or phone, you can add them manually in the donor profile. Select Edit, and then click in Update next to SARS details.

•See the video tutorial below for more detail.
