Prevent late submissions from applicants who are still in phase 2 but have not yet submitted.

Update the application status on the closing deadline.

Go to Development > Projects > Search Advanced and find all applications on Online application phase 1.

Under Project status, select:

  • Workflow phase - Project request - Online application phase 1 
  • tick the Current status only box 
  • Decision date - From - set to the date your applications opened 
  • and select Add to results.

Select Bulk update

Select Workflow 

Generate milestone: 

  • Declined - Incomplete application 
  • Leave Disable workflow box ticked 
  • Proceed. 

Try it out!

Click to

 close.

If your application also has Phase 2 complete status, repeat these steps searching on Phase 2 complete, current status only.

All incomplete applications have been closed.
