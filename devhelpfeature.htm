DevHelp features and how to use it

       Help - features left side

 

•Breadcrumb trails
Breadcrumb navigation shows the user's path to their current location.

•Mail feedback
Activates a mail button in the topic header with which users can mail you feedback on the current topic. References to the topic in question are automatically inserted.

•Print button
Activates a button in the topic header for printing the current topic. This is particularly useful for WebHelp, where some browsers have trouble printing just the topic in a frame.

•Toggles button
Adds a button to the navigation area in the topic header that allows the user to expand and collapse all toggles on the current page by clicking on the button.

•TOC Show/Hide
Adds a slide-out button to the left of the topic title in the topic header that allows the user to hide the entire navigation pane. If the user has cookies enabled the current user-set width of the navigation pane will be remembered between topics. Otherwise the standard width set in the template will be used.
