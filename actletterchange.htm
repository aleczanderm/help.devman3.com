Change to a different letter


Contents

 

The system provides for standardised letters to be generated at certain key point (milestones) during the life-cycle of a project or donation.

 

For example when you first capture a proposal the system allows for an "acknowledgment" letter to be produced.

 

You may decide the standard acknowledgment letter is not appropriate in a particular case. You can change to another acknowledgment letter.
