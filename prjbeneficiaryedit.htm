Beneficiaries of a project can be viewed in a number of different ways, however you can view all of your beneficiaries by year, or by benefactor via the project menu.

 

1.Go to Projects > Beneficiaries > View details

 

2.To reduce the number of records displayed in the grid you can change the filters.
Select a Time period, a Benefactor and / or a Status.  
To view a specific beneficiary record, or to change the beneficiary status, click on the record you're looking for.

 

3.Now you can view and/or make changes to the Beneficiary's details.

 

To change the status, click  on "Change" .

 

4.Select the new status from the Phase drop-down menu, perhaps add some comments in the Comments field e.g. a decline reason. Save > Save and Close.

 

5.In the bursary details screen, the option to send a letter will now appear.  To find out more information about sending letters, see Letters.
