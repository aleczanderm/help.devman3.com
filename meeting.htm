Link a project to a meeting

To link a project to the meeting:

•        Go into a project profile, select Add > Link to a meeting

•        On the bulk update page – select ‘link to a meeting’, then use the search icon to find the meeting from the drop-down list, then select ‘begin process’

•        When it says ‘Successful’ the linking process is complete.

•        You can see the linked meeting on the project summary page

 

Other ways to add projects to a meeting:

•        View project grid (from search page or just view grid)

•        Use check boxes to select specific projects. Select Add to a meeting.

•        Complete the bulk update page as per steps below.
