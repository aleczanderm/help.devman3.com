Merge contacts


Contents

Overview

 

If you have duplicate records of a person or organisation, these need to be merged to combine the history and linked records, such as activities, projects and financial data.

 

Topics

 

How to merge a duplicate person or organisation.

1.Check which profile has the most up to date information.

2.Open the older profile, select edit and add a star (*) to the organisation name, Save > Save and close. Then select Maintenance > Merge.

 

3.Enter the name in the search box, and select the organisation (or person) profile without the star (*).

4.The summary page will display both records for ease of comparison. Select Merge.

 

 

5.DevMan will confirm that the records have been merged. Select the obsolete contact by selecting Back or by clicking on the name and delete the record.
