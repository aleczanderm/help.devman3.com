•Download the template from the Event summary page - Invitees > Add via > Importer > Download template.

•Copy and paste your data onto the template in the correct columns and save.

•Make sure that all cells are in text format.

Send an invitation

 

Tips for event communication

Before the event:

•Send a "save the date" mailer to your target list.

•Send the invitation, including an RSVP link and optional questionnaire and online update link.

•Send reminder by email or SMSs.

During the event using DevMan Go:

•Send a welcome message via email or SMS once the guest is marked as attended.

•Send an event programme if available.

After the event:

•Send follow up communication such as an event survey.

 

 

Manage invitees

 

View invitees and communicate

 

On the Event summary page, click through on the Target list at Invitee status to view the individuals on your invitee list.

 

 

You now have the option to send emails or SMSs to one, many or all of your invitees on the target list.

Select individuals by ticking the check box, or leave blank to select all.

 

Tips and best practice

Send your first invitation to yourself to check that everything has been set up correctly.

 

 

To send an email invitation to your target list, select Send email.

Select the template, stationery, subject line and questionnaire.

 

 

You can edit your invitation details here, or select your own template that you have created.

When all the details are entered correctly, select Send.

 

 

Invitee status

After the invitations have been sent out, update the status of the target list to Invited.

Click through on the Target list, and then select Change status.

 

 

Select the status, then select Begin process.

 

 

As the invitees respond to the invitation via the RSVP link, their status will automatically be updated to accepted or declined.

If some choose to respond via email or phone, update their status manually by clicking through on the status in the event grid, change the status and Save.

 

 

Click through on an invitee's name to open this page and add other pertinent information and special requests.

Invitees who have responded online will be able to record these requests themselves.

 

 

When arriving at the event, the guest status can be updated to Attended using DevManGo.

View the status of your invitees on the Events summary page.

 

 

Extract a list

To view or print a list of your guests:

•Click through on the guest status.

•Select the guest status that you want to view.

•Select Report to view a statistical report or list.

•Select Export to view the list along with special requests.

 

 

Event status

After the event, change the status to Complete so that it will no longer appear in the Events in progress grid.

On the Event summary page, select Edit > Update event status.

Event status can be:

•Planning

•Open

•Closed for RSVP

•Complete
