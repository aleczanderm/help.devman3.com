Overview

 

Use this feature to manage your contract agreements.

Upload, store and organise all your contracts, from donors and suppliers to faculties, etc.

 

Topics

 

Features

•Store all your contracts in a central location.

•Record contract start and end dates.

•Track due dates for donor-related progress reports.

•Set useful reminders for the renewal of contracts or report deadlines.

 

Add

Select Initiatives > Contracts > Add new contract.

•Search for the donor organisation or service provider profile, and select.

•Link to the relevant department, and add the rest of the details.

 

 

Select Save and stay on page.

•Select Add to upload a copy of the contract.

•Select Update to change the status of the contract.

 

 

 

View

View the list of contracts in grid format - select Initiatives > Contracts > View contracts.

 

Manage workflow

Set up workflow triggers at Initiatives > Contracts > Settings > Contract workflow.
