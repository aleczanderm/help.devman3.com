Letter printing problems

 Top 

 

Printing of letters uses more complex technology than that used to print lists and reports.

This is because of the "MS-Word" style formatting where your letters and other fancy documents

When there is a problem with printing you or your IT follow these steps:

 

Logon to the server where Devman and KDE (Kidz Document Engine) is installed.

 

1.Goto Start --> Run and type "Services.msc" and click "OK" as shown here

 

 

The services program will now be open.

This will enable you to make sure the KDE (Kidz Document Engine) service for letters to print is running.

 

2.Locate it, right-click and start or restart it.

Letter printing should now be successful

 

 

If KDE fails to start then check if Winword.exe is running and if so stop it and try KDE start/restart again.

You may need to restart the Devman server
