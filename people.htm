Overview

 

This section will assist you to capture, save and edit your contacts' details and profile.

Links can be added to other profiles on the database.

 

Tips and best practice

DO enter as much accurate information as you can as this will save you time later.

DO check whether a person is already on the system so as to avoid duplications.

 

Topics

 

Add a person

1. On your Home dashboard, select the People menu.

Check whether the person you want to add is on the system by entering the first few letters of the name into the Search for a person box.

 

 

2. If there are no matching results, select Add new person.

Enter as much information as you have, and make sure that it is accurate.

 

 

3. Select Save and stay on page and you will be able to enter additional data as required. Select Save and close once you have completed your entry.

 

 

View a person profile

On your Home dashboard, select the People menu, enter the first few letters of the person's name and then select the correct person from the drop-down list to open the profile page.

Edit a person profile

You can review and change the details of a person, or add additional information.

Open the person's profile summary page, and select Edit.

 

       Person edit page

 

You can add a photograph of your contact to their profile.

Update the profile as needed, then select Save > Save and close to go back to the profile summary page.

To continue working, select Save > Save and stay on page.

You can also select Summary to go back to the person summary page.
