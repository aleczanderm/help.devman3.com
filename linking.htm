Link a person, organisation or project

 

Relationship links can be added and defined between people, organisations and projects.

These can be defined as employee / employer, or beneficiary / benefactor, project contact person, partner organisation etc.

The Employment link allows the person and organisation to be linked, or can be configured to just keep a track of the person's employment record.

Relationship descriptions are also configurable. Please contact the support desk for assistance.

 

Tips and best practice

•When person and organisation details are added via the organisation page, the profiles are automatically linked.

•If a link is deleted, the organisation and person profiles will NOT be deleted.

 

Topics

 

  Link a person to an organisation.

1.On the person profile page select Add > Link > Employment.

 

 

2.To just keep a record of the employment of the person, add all details here, but do not select Organisation on Devman.

 

 

3.To link the person to an organisation that is on Devman, simply check the box Organisation on Devman.

Type in the first few letters of the organisation in the search box. Select the correct organisation, then add all the details that you have.

Then select Save and close.

 

 

  Link an organisation to a person.

1.On the organisation profile page, select Add > Link > Person.

 

 

2.Use the search box to search for the person by typing in the first few letters, and then selecting the correct one.

Add all the details you have on the relationship between the person and the organisation. Select Save & close.

 

 

Add links to a project

1.In the project profile page, select Add > Link > Partnership or Person.

 

 

2.Select the organisation or person you wish to link the project to by entering the first few letter of the name.

Then select Save > Save and close. Your link has been created.

 

 

3.To change the contact person of a project, search for and select the relevant proposal / project, then select  Edit.

Highlight and clear the existing name.

Enter the name of the new contact and select from the dropdown list.

Make any other changes and select Save and close.

 

 

 

Change primary contact

1.From the organisation profile, select View > Links > People.

 

 

2.To change the primary contact person, select the current primary contact, and then deselect the Is this the Person's primary employer? check box.

Select Save and close.

 

 

3.Click on the person who is the new primary contact, and then select the Is this the Person's primary employer? check box.

Select Save and close.

 

 

View links

To view all the people who are linked to the organisation,  select View > Links > People on the organisation profile page.

To view all the organisations, people or projects a person is linked to, select View > Links > Organisations / People / Projects from the person profile page.

To view all the people or partnerships linked to a project, select View > Links > People / Partnerships from the project page.

 

 

The grid displayed shows all the people linked to the organisation, their department, as well as who the primary contact is.

 

 

Edit links

1.From the person, organisation or project page, select View > Links. (See View links ).

2.Select the link you want to edit, make the changes you want, then select Save > Save and close.

3.You can also delete the link altogether by selecting Delete.
