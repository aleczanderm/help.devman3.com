Printing project reports


Contents

 

1.Go to Project > Print.

 

Click here to view

 

2.You now need to choose what information you wish to include in your report.  
 
Project status: You may wish to print a list of only approved projects, in which case you would select the Current and Filter statuses. You can print a report for certain amounts or for a certain time period.
 
Project profile: You can print a report on projects that occur in a particular region, pertain to a certain focus area, or category.
 
For the report below we would like to print a report of all Approved projects from 1 July 2009 to 30 June 2010, for South Africa.

 

Click here to view

 

3.Once you have selected your set of project records, click Search.  

 

Click here to view

 

4.The number of projects that fit your selected criteria will display. If the number of records seems correct to you, you can either View list, to display a grid of your records, or you can continue to the next step of your report printing by clicking Print.

 

Click here to view

 

5.You now need to select how you want your information to be presented i.e. the layout.  There are a number of styles and layouts on the system from basic lists, to statistical information. We suggest that you try the options available and see which report layout best suits your needs.
 
We explain how to print a statistical report with a graph and how to print a project list.
